## COMPATIBILITY SCORING BREAKDOWN

### How the Algorithm Calculates Your Match %

```
BASE SCORE: 50 points

SHARED INTERESTS:
├─ Each shared interest: +10 points
└─ Example: 3 interests = +30 points

RELATIONSHIP ALIGNMENT:
├─ Same relationship goals: +5 points
└─ Example: Both want "Long-term" = +5 points

VALUES ALIGNMENT:
├─ Shared core values: +5 points
└─ Example: Both value "Family" = +5 points

INTENT ALIGNMENT: ⭐ NEW!
├─ Looking for same thing: +10 points
└─ Example: Both want "Casual dating" = +10 points

PERSONALITY COMPATIBILITY: ⭐ NEW!
├─ Similar Big Five traits: +8 points
└─ Example: Both high in Openness = +8 points

RANDOM FACTOR:
└─ Variation: 0-15 points (keeps it interesting!)

FINAL SCORE: Min 40% - Max 99%
```

### Example Calculation:

```
Sarah & John's Match:

Base: 50 pts
+ Shared interests (Hiking, Cooking, Travel): +30 pts
+ Same relationship goals (Long-term): +5 pts
+ Shared values (Family, Health): +5 pts
+ Aligned intent (Long-term): +10 pts ⭐
+ Compatible personalities (3/5 traits): +8 pts ⭐
+ Random factor: +7 pts
─────────────────────────────
TOTAL: 115 pts → Capped at 95%
```
